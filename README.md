# DSE Live — Advanced Market Intelligence Platform

Real-time Dhaka Stock Exchange analysis with 30+ technical indicators.

## Features
- Live stock prices from DSE (via CORS proxy)
- 30+ technical indicators (RSI, MACD, Stochastic, Bollinger, CCI, MFI, ADX, ATR, etc.)
- Candlestick pattern detection
- Smart Top 30 stock selection with sector diversity
- Risk analysis (Sharpe proxy, max drawdown, risk/reward)
- Interactive charts (Chart.js)
- Market heatmap
- 100% client-side — runs on GitHub Pages
